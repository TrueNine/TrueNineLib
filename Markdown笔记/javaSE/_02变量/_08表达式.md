# 表达式
> 以 ; 结尾的 内容,就是一句完整的表达式,(;本身也是)

## ; 结尾的完整表达式

```text
表达式是由 变量,操作符,方法调用组成的结构
```

```java
int i = 5;
System.out.println(i);
```

## ;
> ; 本身也是完整的表达式

```java
public class HelloWorld {
    public static void main(String[] args) {
        //一个空;也是一个表达式
        ;
        ;
        ;       
        ;
    }
}
```
