# 注释
> 在 java 程序中添加注释,有助于理解程序

* 有助于理解,提升代码可读性
* IDE(集成开发环境),自动生成 javadoc 文档

***编译器在编译的时候,会忽略注释部分***

## 单行注释
> //

```// 描述语句 ```

```text
在方法内使用
有利于理解单句程序
```

### 单行注释

```java
// 单行注释
// 单行注释
```

## 多行注释
> /* */

``` /* 描述内容 */ ```

1. 书写多行注释
2. 在语句内书写注释
3. 与 单行注释效果相差无几

### 可代替单行注释(不建议)

```java
/* 这是单行注释 */
```

### 书写多行注释

```java
/* 
	第一行
	第二行
	第三行
*/
```

### 在语句内注释

```java
int a = /* 此处必须是 = */ 1024;
```

## 文档注释
> 书写在类,接口,枚举,类属性,方法.... 上,可协助 IDE生成 javadoc

### 文档注释

```java
/**
 * 文档注释: 这是一个类
 * 用于........
 * 怎么怎么怎么怎..........
 * 
 */
class Test {
	
}
```

# 注意

* 程序员必须练的一手好注释
* 尽管代码写的再烂,也要写好注释
