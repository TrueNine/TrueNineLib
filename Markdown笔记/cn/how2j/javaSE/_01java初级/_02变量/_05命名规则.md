# 命名规则

```text
变量名只能使用:
    字母,数字,$,_
第一个字符只能使用
    字母,$,_
    变量名第一个不能使用数字 

使用完整的单词命名,而非缩写
    比如 moveSpeed,name
不能使用关键字
    但可以包含关键字
        比如: ints,clazz
中文可以命名
    但是极其不建议
```

# 练习 命名规范

```text
思考如下变量命名是否合法，如果不合法，为什么？ 
注: 不要一来就在eclipse中进行验证，因为eclipse几乎已经告诉你答案了
1. int a_;
2. int a@;
3. int a3;
4. int 8@;
5. int 9_;
6. int X$_;
7. int y;
8. int _$_;
9. int $_$;
10. int $*$;
11. int $1$;
12. int _1_;
13. int _@_;
14. int a#;
15. int a";
16. int 123a";
17. int 123a_;
18. int $123b_; 
```

## 答案

```text
1. int a_; // 可以
2. int a@; // 不行,符号内不包括@
3. int a3; // 可以
4. int 8@; // 不行,数字不能在最前
5. int 9_; // 不行,数字不能在最前
6. int X$_; // 可以
7. int y; // 可以
8. int _$_; // 可以
9. int $_$; // 可以
10. int $*$; // 不行, * 是运算符
11. int $1$; // 可以
12. int _1_; // 可以
13. int _@_; // 不行,符号不包括@
14. int a#; // 不行,符号不包括#
15. int a"; // 不行,符号不包括"
16. int 123a"; // 不行,数字不能在最前
17. int 123a_; // 不行,数字不能在最前
18. int $123b_;  // 可以
```
